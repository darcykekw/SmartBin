SmartBin Progress Update

April 30, 2026

I am submitting this progress report to provide an update on my SmartBin HCI project. Currently, the entire software infrastructure is fully complete and operational. I have successfully built the Next.js web dashboard and integrated it with a MySQL database, allowing for real-time monitoring of bin statuses and activity logs. Furthermore, all of the Arduino C++ logic for the ESP32 microcontroller is finished. The code is fully prepared to process ultrasonic sensor data, provide immediate visual and auditory feedback via LEDs and buzzers, and transmit updates directly to the web API.

The only remaining phase is the physical assembly, which is currently delayed due to hardware availability. Since the specific ESP32 board and RGB modules required for this project are not in stock at local electronic shops, I had to order them online and am currently waiting for the delivery. Once the components arrive, I will immediately wire the breadboard, flash the code, and conduct the mandatory usability testing sessions to finalize the project for my final submission.
